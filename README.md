# 🎓 Student Placement Prediction System

##  Internship Task 9 – End-to-End Data Science Project  
### Synent Technology – Data Science Internship

---

##  Project Overview

This project is an End-to-End Data Science application developed to predict whether a student will be placed or not based on academic performance and skill-related attributes.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning Model Building
- Model Evaluation
- Streamlit Deployment

---

##  Objective

The main objective of this project is to build a machine learning model that can predict student placement status using features such as:

- CGPA
- Internships
- Training
- Communication Level
- Technical Courses
- Backlogs
- Innovative Projects

---

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

---

##  Dataset Information

The dataset contains engineering student academic and skill-related details.

### Features:
- Gender
- 10th & 12th Marks
- Engineering Stream
- CGPA
- Internship Status
- Training Status
- Communication Level
- Technical Course
- Placement Status

---

##  Project Workflow

### 1️⃣ Data Collection
- Imported dataset using Pandas

### 2️⃣ Data Cleaning
- Removed unnecessary columns
- Handled missing values
- Encoded categorical values

### 3️⃣ Exploratory Data Analysis
- Analyzed placement trends
- Visualized feature relationships
- Generated correlation heatmaps

### 4️⃣ Model Building
- Used Logistic Regression algorithm
- Split dataset into training and testing data
- Trained and evaluated the model

### 5️⃣ Deployment
- Deployed the model using Streamlit
- Built a simple user interface for prediction

---

##  Key Insights

- Students with higher CGPA have better placement chances
- Internships and training positively impact placements
- Communication skills play an important role
- Backlogs reduce placement probability
